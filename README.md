# 🎉 Youtube Downloader v1.0.0

Nous sommes ravis d'annoncer la première version stable de **Youtube Downloader** ! Une application Windows légère et minimaliste pour télécharger votre musique préférée en haute qualité.

## ✨ Fonctionnalités Principales
*   **Audio Haute Qualité** : Télécharge automatiquement la meilleure qualité audio disponible (MP3).
*   **Support Playlists** : Téléchargez une vidéo unique ou une playlist entière en un clic en fichier MP3 !
*   **Métadonnées Complètes** : Ajoute automatiquement le titre, l'artiste et la pochette au fichier MP3.
*   **Portable** : Un seul fichier `.exe`. Aucune installation ni configuration requise. Transportable sur clé USB.
*   **Interface Épurée** : Design minimaliste (Thème Clair) pour une utilisation simple et rapide.

## 🚀 Comment l'utiliser ?
1.  Téléchargez le fichier `Youtube Downloader.exe` ci-dessous.
2.  Lancez l'application.
3.  Collez un lien YouTube.
4.  Cliquez sur **Download**.
5.  Choisisez le dossier final !

## 🛠 Informations Techniques
*   **Version** : 1.0.0
*   **Développeur** : Danny Berger
*   **Moteur** : Basé sur `yt-dlp` et `ffmpeg` pour une fiabilité maximale.
