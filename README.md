#  Youtube Downloader v1.2.0

**Votre musique, simplement.**

Youtube Downloader est une application Windows simple, rapide et élégante pour télécharger vos morceaux et playlists préférés depuis YouTube en haute qualité. Entièrement réécrite en JavaScript pour des performances optimales, la dernière version apporte une toute nouvelle interface et des fonctionnalités intelligentes pour organiser votre bibliothèque musicale sans effort.
---

## ✨ Pourquoi choisir Youtube Downloader ?

*   **🚀 Rapide et Efficace** : Profitez d'un moteur JavaScript optimisé et de téléchargements parallèles (jusqu'à 10 fichiers en même temps) pour récupérer vos playlists en un temps record.
*   **✨ Interface Moderne** : Une expérience utilisateur soignée et intuitive grâce à une interface graphique entièrement repensée.
*   **🎶 Qualité Audio Supérieure** : Télécharge toujours le meilleur format audio disponible et le convertit en MP3 pour une compatibilité maximale.
*   **🗂️ Organisation Automatique** : Fini le désordre ! Les titres sont automatiquement nommés avec le numéro de piste et d'album lors du téléchargement de playlists.
*   **🏷️ Métadonnées Intelligentes** : Récupère et intègre automatiquement le titre, l'artiste, la pochette et le nom de l'album pour une bibliothèque parfaitement organisée.
*   **🏃 Totalement Portable** : Pas d'installation compliquée. Un seul fichier `.exe` à lancer. Emportez-le partout avec vous sur une clé USB.

---

## 🚀 Comment l'utiliser ?

1.  **Copiez** l'URL d'une vidéo ou d'une playlist YouTube.
2.  **Collez** l'URL dans l'application.
3.  **Cliquez** sur "Télécharger".
4.  C'est tout ! Vos fichiers MP3 apparaissent dans le dossier de votre choix, parfaitement tagués.
